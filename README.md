<h2> Base de Datos - Sistema de Gestión Escolar</h2>

<p>
  Proyecto de base de datos relacional desarrollado en mysql para
  administrar la información de una institución educativa. El sistema organiza y relaciona
  alumnos, docentes, grados, materias, encargados y usuarios mediante claves primarias y
  foráneas, garantizando la integridad de los datos.
</p>

<h3>Tablas principales</h3>

<ul>
  <li><strong>Alumnos:</strong> Información personal, grado asignado y encargado responsable.</li>
  <li><strong>Docentes:</strong> Registro de profesores y la materia que imparten.</li>
  <li><strong>Encargados:</strong> Datos de los responsables de los estudiantes.</li>
  <li><strong>Grados:</strong> Administración de los niveles académicos.</li>
  <li><strong>Materias:</strong> Catálogo de asignaturas impartidas en la institución.</li>
  <li><strong>Usuarios:</strong> Gestión de cuentas y permisos para el acceso al sistema.</li>
</ul>
